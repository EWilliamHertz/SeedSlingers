# Free 2D Game Asset Resources

A curated list of the best sources for free, legally-usable 2D game assets.

## 🌟 Top Recommended Sources

### 1. OpenGameArt.org
**URL:** https://opengameart.org

**Why it's great:**
- Largest curated collection of free game art
- Clear license filtering (CC0, CC-BY, CC-BY-SA, GPL, etc.)
- Active community with quality submissions
- Every asset has clear attribution info

**License Types:**
- CC0 (Public Domain) - No attribution needed
- CC-BY 3.0/4.0 - Attribution required
- CC-BY-SA 3.0/4.0 - Attribution + share-alike
- GPL - Source code must be available

**Best For:**
- Complete sprite collections
- Tileset packs
- Character sprites
- Enemy/monster sprites
- UI elements

**Search Tips:**
- Use filters: License, Tags, Art Type
- Sort by "Most Favorited" for quality
- Look for "asset pack" or "complete" in titles
- Check author's other submissions

**Already Using in SeedSlingers:**
- Hyptosis Tile Collection (CC-BY 3.0)
  - 32×32 and 16×16 pixel art
  - Fantasy RPG environments
  - Monsters, items, interiors, exteriors

### 2. itch.io Game Assets
**URL:** https://itch.io/game-assets/free

**Why it's great:**
- Huge modern collection (3,200+ free tilesets alone)
- Many professional indie artists
- Mix of pixel art, vector, and hand-drawn
- Easy filtering by style, theme, format

**License Types:**
- Varies by creator (ALWAYS check!)
- Many are "Royalty Free" with attribution
- Some CC0, some custom licenses

**Best For:**
- Modern pixel art styles
- Complete game kits
- Themed asset packs
- Character animation packs

**Popular Free Packs:**
- **Mystic Woods** - 16×16 fantasy tileset
- **Ninja Adventure** - Top-down action pack
- **Sprout Lands** - Cozy farming assets
- **Brackeys Platformer Bundle** - Complete starter kit
- **Kings and Pigs** - Platformer characters
- **Tiny Swords** - Battle arena pack

**Search Tips:**
- Filter by "Royalty Free" tag
- Check "No AI" filter for human-made art
- Look at creator's profile for more packs
- Read description for license details

### 3. Kenney.nl (Kenny Assets)
**URL:** https://kenney.nl/assets

**Why it's great:**
- ALL assets are CC0 (Public Domain)
- No attribution required!
- Consistent, professional quality
- Huge variety of styles and themes

**License:** CC0 - Use however you want!

**Best For:**
- Prototyping (no license worries)
- UI elements
- Simple, clean sprites
- 3D assets too

**Asset Categories:**
- Pixel Art
- 2D Sprites
- UI Packs
- Isometric
- 3D Models
- Audio

**Why CC0 is Amazing:**
- Use commercially without credit
- Modify freely
- No licensing headaches
- Perfect for prototypes

### 4. CraftPix.net Freebies
**URL:** https://craftpix.net/freebies

**Why it's great:**
- Professional commercial-quality art
- Well-organized by theme/style
- Regular new releases
- Both free and premium options

**License Types:**
- Usually Royalty-Free (check each pack)
- May require attribution
- Some allow commercial use

**Best For:**
- Polished production assets
- Top-down RPG tilesets
- Platformer sprite packs
- UI themes

**Asset Types:**
- Pixel Art Tilesets
- Character Sprites
- Vector Graphics
- GUI Packs
- Effects/VFX

### 5. Game-Icons.net
**URL:** https://game-icons.net

**Why it's great:**
- 4,000+ SVG game icons
- All CC-BY 3.0
- Searchable database
- Customizable colors

**License:** CC-BY 3.0 (Attribution required)

**Best For:**
- Item icons (weapons, potions, gear)
- Ability/skill icons
- Status effects
- UI elements
- Inventory systems

**Features:**
- Download as SVG or PNG

- Search by keyword

## 📋 Quick License Guide

### CC0 (Public Domain)
✅ Use for anything  
✅ Modify freely  
✅ No attribution required  
✅ Commercial use  
**Examples:** Kenney.nl, some OpenGameArt

### CC-BY 3.0/4.0
✅ Use for anything  
✅ Modify freely  
⚠️ **Attribution REQUIRED**  
✅ Commercial use  
**Examples:** Hyptosis, Game-Icons.net

### CC-BY-SA 3.0/4.0
✅ Use for anything  
✅ Modify freely  
⚠️ **Attribution REQUIRED**  
⚠️ **Share modifications under same license**  
✅ Commercial use  

### Royalty-Free (Custom)
✅ Usually commercial use  
⚠️ Read specific terms  
⚠️ May require attribution  
**Examples:** Many itch.io assets

### GPL (GNU General Public License)
✅ Use freely  
✅ Modify freely  
⚠️ Must make source code available  
⚠️ Derivative works must be GPL  
✅ Commercial use

## 🔍 How to Evaluate Assets

### ✅ Good Signs
- Clear license information
- Author actively responds to comments
- Multiple downloads/favorites
- Source files included (.aseprite, .psd)
- Preview shows all assets
- Compatible dimensions (16×16, 32×32, etc.)

### 🚩 Red Flags
- No license mentioned
- "Free for personal use only" (can't publish)
- Watermarked images
- No attribution information
- Suspiciously professional (might be stolen)
- Inconsistent style within pack

## 🎯 Finding Assets for Your Game

### Step 1: Define Your Needs
```
□ Pixel art size? (16×16, 32×32, 64×64)
□ Art style? (Retro, modern, minimalist)
□ Theme? (Fantasy, sci-fi, modern)
□ What assets? (Tiles, characters, items)
□ License requirements? (CC0, CC-BY ok?)
```

### Step 2: Search Smart
- Start with OpenGameArt.org for variety
- Check itch.io for modern styles
- Use Kenney.nl for quick prototyping
- Game-Icons.net for UI icons

### Step 3: Test Compatibility
- Download preview/sample
- Test in your game engine
- Check style consistency
- Verify license allows your use

### Step 4: Document Everything
- Save original download URLs
- Note author names
- Record license type
- Keep attribution text

## 📝 Attribution Best Practices

### In Your Game:
1. **Credits Screen** - List all artists
2. **About/Help Menu** - Link to credits
3. **README.md** - Full attribution list
4. **Website** - Credits page

### Attribution Format:
```
Asset Name by Artist Name
License: CC-BY 3.0
Source: [URL]
```

### Example:
```
Hyptosis Tile Collection by Hyptosis
License: CC-BY 3.0
Source: https://opengameart.org/content/lots-of-free-2d-tiles-and-sprites-by-hyptosis
```
